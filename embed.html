FormEmbed SDK - Complete Documentation

Embed your JSONPost forms anywhere with a simple JavaScript snippet.

Installation

Add the SDK to your HTML:

<script src="https://forms.jsonpost.com/embed/v1/embed.js"></script>

Quick Start Examples

1. Modal Popup

Opens a centered modal dialog with your form:

<script>
  const modal = FormEmbed.createModal('ec14584c-be46-486d-bb51-1733801d7bc3', {
    theme: 'bumblebee',
    width: 800,
    height: 600
  });
  
  modal.open();
</script>

2. Chatbox / Popover

Floating chat icon that opens a popover window:

<script>
  FormEmbed.createChatbox('ec14584c-be46-486d-bb51-1733801d7bc3', {
    theme: 'bumblebee',
    position: 'bottom-right',
    buttonIcon: 'https://your-icon.png',
    buttonColor: '#3a7685',
    width: 400,
    height: 600
  });
</script>

3. Side Drawer

Slides in from left or right side:

<script>
  FormEmbed.createDrawer('ec14584c-be46-486d-bb51-1733801d7bc3', {
    theme: 'bumblebee',
    side: 'right',
    width: 500,
    buttonText: 'Feedback',
    buttonColor: '#3a7685'
  });
</script>

4. Button Trigger

Trigger modal or drawer from any button:

<button id="my-button">Open Survey</button>

<script>
  FormEmbed.createButton('ec14584c-be46-486d-bb51-1733801d7bc3', {
    theme: 'bumblebee',
    buttonSelector: '#my-button',
    openAs: 'modal',
    width: 800,
    height: 600
  });
</script>

Mobile Behavior

Important: On mobile devices (width < 768px), all embed modes automatically convert to fullscreen modals. Borders and rounded corners are removed for edge-to-edge display.

API Methods

Modal Methods

const modal = FormEmbed.createModal(formId, options);

modal.open();    // Open the modal
modal.close();   // Close the modal
modal.toggle();  // Toggle open/closed state

Chatbox Methods

const chatbox = FormEmbed.createChatbox(formId, options);

chatbox.open();    // Open the chatbox window
chatbox.close();   // Close the chatbox window
chatbox.toggle();  // Toggle open/closed state
chatbox.destroy(); // Remove chatbox and button from page

Drawer Methods

const drawer = FormEmbed.createDrawer(formId, options);

drawer.open();    // Open the drawer
drawer.close();   // Close the drawer
drawer.toggle();  // Toggle open/closed state
drawer.destroy(); // Remove drawer and button from page

Troubleshooting

Form not loading?

  • Verify your form ID is correct
  • Check browser console for errors
  • Ensure the SDK script loads before your code
  • Check if your page blocks iframes (CSP headers)

Modal not centered?

  • Try percentage-based dimensions: width: '80%'
  • Check for CSS conflicts (z-index, position)
  • Verify no parent elements have overflow: hidden

Chatbox button not appearing?

  • Check if z-index conflicts exist
  • Verify button icon URL is accessible (CORS)
  • Check browser console for errors
  • Ensure SDK is loaded

Drawer not sliding in?

  • Check for CSS transitions being disabled
  • Verify no conflicting styles on body
  • Test in different browsers

Best Practices

  • Load SDK once: Include the script tag only once per page
  • Wait for DOM: Initialize embeds after DOM is ready or use defer
  • Mobile-first: Test on mobile - embeds auto-adapt to fullscreen
  • Theme consistency: Match form theme to your site design
  • Performance: Lazy load embeds (don't auto-open immediately)
  • Accessibility: Provide keyboard navigation (ESC to close)
  • User experience: Don't show modal immediately on page load

Browser Support

  • Chrome/Edge (latest 2 versions)
  • Firefox (latest 2 versions)
  • Safari (latest 2 versions)
  • Mobile browsers (iOS Safari, Chrome Mobile)

Security & Privacy

  • Forms load in sandboxed iframes
  • No cookies or localStorage used by SDK
  • HTTPS-only form loading
  • No tracking or analytics in SDK
